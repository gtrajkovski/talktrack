# TalkTrack — Current State

## App Status: Feature Complete, Uncommitted Changes

TalkTrack is a voice-first rehearsal coach for presentations. The core promise: complete an entire rehearsal session using ONLY voice after one initial tap.

### What's Built

**Core Rehearsal Modes (all voice-controllable):**
- ✅ Listen Mode — TTS reads slides, auto-advances
- ✅ Prompt Mode — Title only, user recites, can reveal answer
- ✅ Test Mode — Title only, scored against original notes
- ✅ Completion Screen — "again" or "done" via voice

**Voice Commands (all modes):**
- next, back, repeat, stop, resume
- reveal (Prompt), help (Test)
- again, done (Completion)

**Voice-First UX Features:**
- ✅ Earcons — 11 Web Audio API sounds for state changes
- ✅ StateOrb — 120px glanceable pulsing indicator
- ✅ Progressive hints — learns which commands you use
- ✅ Patience mode — 15s nudge, exponential backoff on errors
- ✅ Streaks — daily practice tracking with TTS encouragement

**Import Options:**
- ✅ Paste text (splits on blank lines)
- ✅ File upload (PPTX speaker notes, DOCX)
- ✅ Demo talk for first-time users

**Additional Features:**
- ✅ Multi-language voice commands (EN, MK, SQ, IT)
- ✅ Audio recording per slide (MediaRecorder API)
- ✅ Progress heatmap on stats page
- ✅ Timer overlay with warning threshold
- ✅ Section-based rehearsal
- ✅ Session pause/resume
- ✅ Onboarding modal
