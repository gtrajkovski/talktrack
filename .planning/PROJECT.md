# TalkTrack

## What This Is
A mobile-first, voice-first web app that lets people rehearse spoken content (presentations, speeches, pitches) entirely hands-free and eyes-free. The first rehearsal tool designed for commuters.

## Core Value
Rehearse your talk using only your voice and ears — no screen required.

## Tech Stack
- Next.js 15 (App Router), TypeScript, Tailwind CSS v4
- Zustand for state, IndexedDB via idb for persistence
- Web Speech API for TTS and recognition
- Web Audio API for chime generation
- 100% client-side — zero external API calls

## Status
MVP in progress — Phase 1 (paste-to-rehearse flow with Listen/Prompt/Test modes)
