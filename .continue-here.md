<remaining_work>

**To build APK (requires Android development environment):**
1. Install Android Studio or JDK 17+
2. Set JAVA_HOME environment variable
3. Run: `cd android && ./gradlew assembleDebug`
4. APK will be at: `android/app/build/outputs/apk/debug/app-debug.apk`
5. Install on device and test audio continues when screen locks

**Alternative - Open in Android Studio:**
1. Run `npx cap open android`
2. Build and run from Android Studio

</remaining_work>
